## How to Contribute

1.  **Fork the repository** on GitHub.
2.  **Clone the project** to your own machine.
3.  **Create a branch** for your feature or fix:
    ```bash
    git checkout -b feature/amazing-feature
    ```
4.  **Commit your changes** (please provide descriptive commit messages).
    ```bash
    git commit -m "Add some amazing feature"
    ```
5.  **Push to the branch**:
    ```bash
    git push origin feature/amazing-feature
    ```
6.  **Open a Pull Request**.

## Coding Standards

*   **TypeScript**: Please ensure there are no compilation errors.
*   **Linting**: Run `npm run lint` (if available) or ensure code style is consistent.
*   **Clean Code**: Keep functions small and meaningful.

## Reporting Bugs

Please check if the issue has already been reported. If not, open a new issue with:
*   Steps to reproduce.
*   Expected behavior.
*   Actual behavior.
