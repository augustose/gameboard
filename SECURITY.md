# Security Policy

## Supported Versions

By default, we only support the latest version of the application.

| Version | Supported          |
| ------- | ------------------ |
| 1.x.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

We take security seriously. If you discover a security vulnerability, please do **NOT** create a public GitHub issue.

Instead, please send an email to [INSERT_CONTACT_EMAIL] or reach out privately. We will assess the issue and provide an estimated timeline for a fix.

### What to include in your report:

*   Description of the vulnerability.
*   Steps to reproduce the issue.
*   Potential impact.

We appreciate your effort to make **El Turix** safer for everyone.
